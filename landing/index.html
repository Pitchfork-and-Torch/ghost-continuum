    Quickstart

    From zero to Command Nexus

    Node.js 18+. Core engine: zero npm dependencies. UI may load Three.js from CDN for WebGL (canvas fallback offline).

    Estimated time · ~5 minutes on a modern machine

    git clone https://github.com/Pitchfork-and-Torch/ghost-continuum.git
cd ghost-continuum
npm run setup
npm start
# → http://127.0.0.1:30000 COMMAND NEXUS

    FAQ

    Questions operators ask

    What is Ghost Continuum?

    An open-source Living Digital Immune System: polymorphic honeypots, edge tripwires, NSGA-II genome evolution, holographic Command Nexus, and Merkle-sealed forensics - for authorized defensive use.

    Is any of this offensive?

    No. Defensive only. Scope probes are defensive presets; exploit roles are blocked; hub binds to 127.0.0.1 by default.

    Does it require the cloud?

    No. Local-first. Cloudflare Edge is optional for sites you already host there. Core brain and data stay on your machine under ~/.ghost-continuum.

    What are the dependencies?

    Core engine: zero npm packages. Node.js 18+ only. UI may load Three.js and fonts from CDN with offline/canvas fallbacks.

    What is new in v3.0 OMEGA ASCENDANT?

    Immersive public Command interface, interactive plane explorer, client-side Nexus lifecycle demo, Home Shield spotlight, efficacy presentation, and polished cockpit/docs packaging - same defensive philosophy.
